# База знаний по предметным областям: Java

Проект-база знаний для подготовки и развития Java-разработчика. Состоит из двух частей.

## Структура

| Раздел | Что внутри |
|--------|-----------|
| [`knowledge-base/`](knowledge-base/README.md) | **Задание 1.** Список предметных областей, которые должен знать современный Java-программист. У каждой области — описание, ключевые подтемы и 3–4 достоверных источника. |
| [`interview-questions/`](interview-questions/README.md) | **Задание 2.** База вопросов для собеседования на Java-разработчика с проверенными ответами (секции «Оригинальный ответ из интернета» и «Ответ от Claude»). Разбита по разделам. |
| [`java-docs/`](java-docs/README.md) | **Java-документация.** Перевод официального руководства [The Java Tutorials](https://docs.oracle.com/javase/tutorial/) (Oracle) на русский: трейл → урок → подтема, диаграммы пересозданы в Mermaid. Наполняется инкрементально. |
| [`webapp/`](webapp/README.md) | Веб-просмотрщик базы (вопросы + области знаний + Java-документация) с поиском, фильтрами, подсветкой кода и Mermaid-диаграммами. Один файл на Python без зависимостей: `cd webapp && python3 app.py`. |

## Быстрый запуск приложения

Из терминала в корне проекта:

```bash
./start-app.sh        # или: cd webapp && python3 app.py
```

Откроется локальный сервер, браузер запустится автоматически (http://127.0.0.1:8000/).
Остановка — `Ctrl+C`. Нужен только Python 3, внешних зависимостей нет.

> Для запуска двойным кликом из файлового менеджера можно создать ярлык `.desktop`,
> прописав в нём абсолютные пути `Exec=` и `Path=` к этому каталогу (ярлык машинозависим
> и в репозиторий не входит).

## Принципы

- Всё на русском языке. Иностранные источники переводятся.
- Источники — реальные и проверенные (приоритет: официальная документация Oracle/Spring/Hibernate, Baeldung, признанные книги и авторы).
- Ответы на вопросы не выдумываются: ищутся в интернете, сверяются по нескольким источникам.
- Файлы перекрёстно ссылаются друг на друга: область знаний ↔ соответствующие вопросы.
