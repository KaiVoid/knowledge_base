# Рабочая документация (для Claude)

Здесь хранится вспомогательная дока: шаблоны и накопительные факт-логи. Цель —
не разбрасывать служебные материалы по проекту (см. правило в памяти проекта).

- `templates/` — переиспользуемые шаблоны контента (копируй и заполняй).
- `<тема>.md` — «живые» факт-логи: пополняются по мере выполнения задач.
- `superpowers/` — планы и спеки superpowers (в `.gitignore`, локально).
