# Фундамент языка и платформы

Базовые вопросы по языку Java и платформе JVM: синтаксис и ООП, коллекции, многопоточность
и concurrency, модель памяти и сборка мусора, generics, исключения, ввод-вывод (IO/NIO),
функциональный стиль и Stream API, современные возможности языка.

## Источники

Вопросы собраны из открытых подборок вопросов с собеседований, переведены на русский и выверены
по нескольким источникам. Конкретный источник указан в каждом вопросе (строка «Источник» в секции
«Оригинальный ответ из интернета»). Теория по этим темам — в группе «Предметные области» вкладки «Теория».
