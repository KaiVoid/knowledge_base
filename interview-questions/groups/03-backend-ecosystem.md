# Backend-экосистема

Вопросы по серверной экосистеме Java: Spring Framework и Spring Boot, реляционные базы данных и SQL
(включая специфику PostgreSQL), NoSQL и кеши (Redis, Apache Cassandra, ClickHouse), JPA/Hibernate
и jOOQ, REST и веб-слой, системы сборки (Maven, Gradle) и тестирование.

## Источники

Вопросы собраны из открытых подборок вопросов с собеседований и выверены по нескольким источникам;
конкретный источник указан в каждом вопросе. Теория по этим темам — в группе «Предметные области» вкладки «Теория».
