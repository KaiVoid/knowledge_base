# Разработка с ИИ

Вопросы по разработке с использованием искусственного интеллекта и агентской разработке:
AI-ассистированная разработка (GitHub Copilot, Cursor, Claude Code), промпт-инжиниринг, основы
больших языковых моделей (LLM), Retrieval-Augmented Generation (RAG), ИИ-агенты и агентские
паттерны, протокол Model Context Protocol (MCP), а также безопасность и оценка LLM-приложений.

## Источники

Вопросы собраны из открытых подборок вопросов с собеседований и официальной/авторитетной
документации (Anthropic, OpenAI, modelcontextprotocol.io, OWASP, GitHub, Hugging Face, LangChain
и др.), переведены на русский и выверены по нескольким источникам; конкретный источник указан
в каждом вопросе.
