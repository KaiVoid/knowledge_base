# Вопросы с HH

Раздел собран из квизов проверки навыков сервиса HeadHunter (hh.ru) — наборов вопросов,
которыми на платформе подтверждают технические навыки. Каждый вопрос переведён и выверен:
есть исходный вариант ответа и «Ответ от Claude». Внутри раздел разбит по темам
(PostgreSQL, Java, Docker, OOP, SQL), внутри тем — по уровням (Advanced / Intermediate / Basic).
