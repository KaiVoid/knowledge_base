# Урок 1. Подготовка к сертификации по языку Java

**Трейл:** Java Certification · **Оригинал:** [Preparation for Java Programming Language Certification](https://docs.oracle.com/javase/tutorial/extra/certification/index.html)
**Связанные области:** [[01-core-java-syntax-oop]] · **Вопросы:** core-java

> Перевод официального руководства Oracle (The Java Tutorials, JDK 8). Страница содержит
> список доступных ресурсов для подготовки к сертификации.

Oracle предлагает два пути сертификации для программистов Java SE 8:

- [Oracle Certified Associate, Java SE 8 Programmer](https://education.oracle.com/oracle-certified-associate-java-se-8-programmer/trackp_333) —
  сертификация уровня «ассоциированный специалист» (OCA).
- [Oracle Certified Professional, Java SE 8 Programmer](https://education.oracle.com/oracle-certified-professional-java-se-8-programmer/trackp_357) —
  сертификация уровня «профессионал» (OCP).

Высокоуровневый ориентир по подходу к изучению языка Java — в материале
[Java Tutorials Learning Paths](https://docs.oracle.com/javase/tutorial/tutorialLearningPaths.html)
(пути обучения).

> **Примечание Oracle.** The Java Tutorials написаны для JDK 8. Примеры и практики на этой
> странице не учитывают улучшения более поздних выпусков и могут использовать технологии,
> которые больше недоступны. Актуальные материалы:
> - [Dev.java](https://dev.java/learn/) — обновлённые руководства с учётом новейших выпусков;
> - [Java Language Changes](https://docs.oracle.com/pls/topic/lookup?ctx=en/java/javase&id=java_language_changes) —
>   сводка изменений языка в Java SE 9 и последующих версиях;
> - [JDK Release Notes](https://www.oracle.com/technetwork/java/javase/jdk-relnotes-index-2162236.html) —
>   сведения о новых возможностях, улучшениях и удалённых/устаревших опциях во всех выпусках JDK.
