# Трейл 20. JNDI (Java Naming and Directory Interface)

**Оригинал:** [Trail: Java Naming and Directory Interface](https://docs.oracle.com/javase/tutorial/jndi/index.html)

Доступ к службам именования и каталогов (например, DNS и LDAP) через единый интерфейс:
концепции, обзор архитектуры, настройка, операции именования и каталога, продвинутые темы LDAP,
доступ к объектам в каталоге и возможности JDK 5/6.
