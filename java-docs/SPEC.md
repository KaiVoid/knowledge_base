# Спецификация раздела «Java-документация»

**Дата:** 2026-06-16 · **Статус:** утверждено, в работе

## Цель

Перенести официальную документацию **The Java Tutorials (Oracle)** в локальную базу
знаний на русском языке. Структура оригинала (трейл → урок → подтема) сохраняется.
Всё, что есть в оригинале, постепенно перерабатывается и переводится. Наполнение —
инкрементальное, по одному трейлу за подзадачу.

## Источник

- The Java Tutorials, Oracle: <https://docs.oracle.com/javase/tutorial/>
- Официальное руководство Oracle — высший приоритет достоверности по правилам проекта.

## Полное оглавление (26 трейлов)

### Базовые (приоритет 1)
1. **01 Getting Started** — феномен Java, Hello World, разбор Hello World, типичные проблемы.
2. **02 Learning the Java Language** — ООП, основы языка, классы и объекты, аннотации,
   интерфейсы и наследование, числа и строки, generics, пакеты.
3. **03 Essential Java Classes** — исключения, базовый I/O, конкурентность, окружение,
   регулярные выражения.
4. **04 Collections** — введение, интерфейсы, агрегатные операции, реализации, алгоритмы,
   свои реализации, совместимость.
5. **05 Date-Time APIs** — обзор, стандартный календарь, конвертация не-ISO дат,
   функциональность даты/времени, legacy-код, итоги.
6. **06 Deployment** — апплеты, Web Start, RIA, deployment in-depth, self-contained, JAR.
7. **07 Java Certification** — ресурсы для подготовки к сертификации.

### Специализированные (приоритет 2)
8. Creating a GUI with Swing · 9. Creating a JavaFX GUI · 10. Custom Networking ·
11. The Extension Mechanism · 12. Full-Screen Exclusive Mode API · 13. Generics (расширенный) ·
14. Internationalization · 15. JavaBeans · 16. JAXB · 17. JAXP · 18. JDBC Database Access ·
19. JMX · 20. JNDI · 21. Reflection · 22. RMI · 23. Security · 24. Sockets Direct Protocol ·

## Структура каталога

```
java-docs/
  README.md                     # индекс: оглавление + статус наполнения
  RULES.md                      # правила оформления переводов
  SPEC.md                       # этот файл
  assets/<trail>/...            # перерисованные/исходные изображения
  NN-<trail-slug>/
    README.md                   # обзор трейла + список уроков
    NN-<lesson-slug>.md         # один урок = один файл
```

## Формат файла урока

- Шапка: трейл, ссылка на оригинал Oracle, связанные области `[[NN-slug]]`, раздел вопросов.
- Тело: перевод текста на русский, диаграммы, переведённые примеры кода.
- Код примеров сохраняется как есть; **комментарии в коде переводятся**.
- Завершающая секция «## Источник» с точной ссылкой на страницу Oracle.

## Графика

- **Диаграммы** (иерархии классов, потоки, декораторы I/O) → пересоздаются в **Mermaid**
  с русскими подписями (версионируется, рендерится в Markdown).
- **Сложный растр**, не ложащийся в Mermaid → перерисовывается с русским текстом в
  `assets/<trail>/`, подключается как `![](...)` с русской подписью.
- Скриншоты IDE/инсталляторов → иллюстративные, с переведённой подписью.

## Интеграция в webapp (один на весь проект)

- В `webapp/app.py`: третий источник `JD_DIR = java-docs`, рекурсивный обход
  (трейл → урок), новый подраздел «Java-документация» в UI.
- Mermaid: спец-обработка ` ```mermaid ` блоков → `<div class="mermaid">`, плюс локально
  вендоренный `mermaid.min.js` (офлайн, без CDN).

## Порядок выполнения

1. Каркас: `java-docs/{README,RULES,SPEC}.md`, структура каталогов. ← готово
2. Правка `webapp/app.py` + вендоринг mermaid.js → подраздел отображается.
3. Трейл 01 (Getting Started) целиком — эталон формата.
4. Трейлы 02→07, затем 08→26 — по одному за подзадачу, с пополнением оглавления.
