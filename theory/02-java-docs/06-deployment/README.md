# Трейл 06. Deployment (Развёртывание)

**Оригинал:** [Trail: Deployment](https://docs.oracle.com/javase/tutorial/deployment/index.html)

Как упаковывать и развёртывать приложения и апплеты с помощью JAR-файлов, Java Web Start и
Java Plug-in.

> **Примечание.** Большинство технологий этого трейла (апплеты, Java Plug-in, Java Web Start)
> устарели и удалены в современных версиях Java (Plug-in — с JDK 11, Web Start — с JDK 11).
