# Урок 5. Стилизация форм с помощью JavaFX CSS

**Трейл:** Creating a JavaFX GUI · **Оригинал:** [Fancy Forms with JavaFX CSS](https://docs.oracle.com/javase/8/javafx/get-started-tutorial/css.htm)
**Связанные области:** [[01-core-java-syntax-oop]] · **Вопросы:** core-java

> Перевод официального руководства Oracle (JavaFX 8).
>
> Этот урок посвящён тому, как сделать ваше JavaFX-приложение привлекательным, добавив каскадную таблицу стилей (Cascading Style Sheet, CSS). Вы разработаете дизайн, создадите файл `.css` и примените новые стили.
>
> В этом уроке вы возьмёте форму входа (Login), в которой для подписей (labels), кнопок (buttons) и цвета фона используются стили по умолчанию, и с помощью нескольких простых изменений CSS превратите её в стилизованное приложение, как показано на Рисунке 5-1.
>
> Инструмент, используемый в этом руководстве «Getting Started», — среда разработки NetBeans IDE. Прежде чем начать, убедитесь, что используемая вами версия NetBeans IDE поддерживает JavaFX 8. Подробности смотрите на странице сертифицированных конфигураций систем (Certified System Configurations) на странице загрузок Java SE Downloads.

## Создание проекта

> Если вы проходили руководство «Getting Started» с самого начала, значит, вы уже создали проект Login, необходимый для этого урока. Если нет — загрузите проект Login, щёлкнув правой кнопкой мыши по файлу Login.zip и сохранив его в вашей файловой системе. Извлеките файлы из zip-архива, а затем откройте проект в NetBeans IDE.

## Создание файла CSS

> Ваша первая задача — создать новый файл CSS и сохранить его в том же каталоге, что и главный класс вашего приложения. После этого нужно сделать так, чтобы JavaFX-приложение «узнало» о добавленной каскадной таблице стилей.
>
> 1. В окне Projects среды NetBeans IDE разверните узел проекта Login, а затем узел каталога Source Packages.
> 2. Щёлкните правой кнопкой мыши по папке login в каталоге Source Packages и выберите New, затем Other.
> 3. В диалоговом окне New File выберите Other, затем Cascading Style Sheet и нажмите Next.
> 4. Введите Login в текстовое поле File Name и убедитесь, что в текстовом поле Folder указано значение `src\login`.
> 5. Нажмите Finish.
> 6. В файле `Login.java` инициализируйте переменную таблиц стилей (stylesheets) класса `Scene` так, чтобы она указывала на каскадную таблицу стилей, добавив строку кода, выделенную ниже жирным, чтобы получился результат, показанный в Примере 5-1.

**Пример 5-1. Инициализация переменной stylesheets**

```java
Scene scene = new Scene(grid, 300, 275);
primaryStage.setScene(scene);
scene.getStylesheets().add
 (Login.class.getResource("Login.css").toExternalForm());
primaryStage.show();
```

> Этот код ищет таблицу стилей в каталоге `src\login` проекта NetBeans.

## Добавление фонового изображения

> Фоновое изображение помогает сделать вашу форму более привлекательной. В этом уроке вы добавите серый фон с текстурой, похожей на лён (linen).
>
> Сначала загрузите фоновое изображение, щёлкнув правой кнопкой мыши по картинке background.jpg и сохранив её в папку `src\login` проекта Login в NetBeans.
>
> Теперь добавьте код для свойства `background-image` в файл CSS. Помните, что путь указывается относительно таблицы стилей. Поэтому в коде из Примера 5-2 изображение `background.jpg` находится в том же каталоге, что и файл `Login.css`.

**Пример 5-2. Фоновое изображение**

```css
.root {
     -fx-background-image: url("background.jpg");
}
```

> Фоновое изображение применяется к стилю `.root`, что означает, что оно применяется к корневому узлу (root node) экземпляра `Scene`. Определение стиля состоит из имени свойства (`-fx-background-image`) и значения этого свойства (`url("background.jpg")`).

## Стилизация подписей (labels)

> Следующие элементы управления, которые нужно улучшить, — это подписи (labels). Вы будете использовать класс стиля `.label`, что означает, что стили затронут все подписи в форме. Код приведён в Примере 5-3.

**Пример 5-3. Размер шрифта, заливка, насыщенность и эффект для подписей**

```css
.label {
    -fx-font-size: 12px;
    -fx-font-weight: bold;
    -fx-text-fill: #333333;
    -fx-effect: dropshadow( gaussian , rgba(255,255,255,0.5) , 0,0,0,1 );
}
```

> Этот пример увеличивает размер и насыщенность шрифта и применяет отбрасываемую тень (drop shadow) серого цвета (#333333). Цель отбрасываемой тени — добавить контраст между тёмно-серым текстом и светло-серым фоном. Подробности о параметрах свойства отбрасываемой тени смотрите в разделе об эффектах в [справочном руководстве JavaFX CSS Reference Guide](https://docs.oracle.com/javase/8/javafx/api/javafx/scene/doc-files/cssref.html#typeeffect).

## Стилизация текста

> Теперь создайте несколько специальных эффектов для двух объектов `Text` в форме: `scenetitle`, который содержит текст `Welcome`, и `actiontarget`, который является текстом, возвращаемым при нажатии пользователем кнопки Sign in. К объектам `Text`, используемым столь по-разному, можно применять разные стили.
>
> 1. В файле `Login.java` удалите следующие строки кода, задающие встроенные (inline) стили, которые в настоящее время установлены для текстовых объектов:
>
>    ```java
>    scenetitle.setFont(Font.font("Tahoma", FontWeight.NORMAL, 20));
>    actiontarget.setFill(Color.FIREBRICK);
>    ```
>
>    Переходя от встроенных стилей к CSS, вы отделяете дизайн от содержимого. Такой подход облегчает дизайнеру управление стилем без необходимости изменять содержимое.
>
> 2. Создайте идентификатор (ID) для каждого текстового узла с помощью метода `setId()` класса Node: добавьте следующие строки, выделенные жирным, чтобы получился результат, показанный в Примере 5-4.

**Пример 5-4. Создание ID для текстовых узлов**

```java
...
Text scenetitle = new Text("Welcome");
scenetitle.setId("welcome-text");
...
...
grid.add(actiontarget, 1, 6);
actiontarget.setId("actiontarget");
..
```

> 3. В файле `Login.css` определите свойства стиля для идентификаторов `welcome-text` и `actiontarget`. В качестве имени стиля используйте ID с предшествующим знаком решётки (#), как показано в Примере 5-5.

**Пример 5-5. Эффект для текста**

```css
#welcome-text {
   -fx-font-size: 32px;
   -fx-font-family: "Arial Black";
   -fx-fill: #818181;
   -fx-effect: innershadow( three-pass-box , rgba(0,0,0,0.7) , 6, 0.0 , 0 , 2 );
}
#actiontarget {
  -fx-fill: FIREBRICK;
  -fx-font-weight: bold;
  -fx-effect: dropshadow( gaussian , rgba(255,255,255,0.5) , 0,0,0,1 );  
}
```

> Размер текста Welcome увеличивается до 32 пунктов, а шрифт меняется на Arial Black. Цвет заливки текста устанавливается в тёмно-серый (#818181), и применяется эффект внутренней тени (inner shadow), создающий эффект тиснения (embossing). Внутреннюю тень можно применить к любому цвету, изменив цвет заливки текста на более тёмную версию фона. Подробности о параметрах свойства внутренней тени смотрите в разделе об эффектах в [справочном руководстве JavaFX CSS Reference Guide](https://docs.oracle.com/javase/8/javafx/api/javafx/scene/doc-files/cssref.html#typeeffect).
>
> Определение стиля для `actiontarget` похоже на то, что вы видели ранее.
>
> На Рисунке 5-4 показаны изменения шрифта и эффекты теней на двух объектах `Text`.

## Стилизация кнопки

> Следующий шаг — стилизация кнопки, чтобы она меняла свой стиль, когда пользователь наводит на неё мышь. Это изменение даст пользователям понять, что кнопка интерактивна, — стандартная практика проектирования.
>
> Сначала создайте стиль для исходного состояния кнопки, добавив код из Примера 5-6. В этом коде используется селектор класса стиля `.button`, так что если вы позже добавите в форму ещё одну кнопку, то и она будет использовать этот стиль.

**Пример 5-6. Отбрасываемая тень для кнопки**

```css
.button {
    -fx-text-fill: white;
    -fx-font-family: "Arial Narrow";
    -fx-font-weight: bold;
    -fx-background-color: linear-gradient(#61a2b1, #2A5058);
    -fx-effect: dropshadow( three-pass-box , rgba(0,0,0,0.6) , 5, 0.0 , 0 , 1 );
}
```

> Теперь создайте немного другой вид для случая, когда пользователь наводит мышь на кнопку. Это делается с помощью псевдокласса (pseudo-class) hover. Псевдокласс включает селектор для класса и имя состояния, разделённые двоеточием (:), как показано в Примере 5-7.

**Пример 5-7. Стиль кнопки при наведении (hover)**

```css
.button:hover {
    -fx-background-color: linear-gradient(#2A5058, #61a2b1);
}
```
