# Трейл 11. The Extension Mechanism (Механизм расширений)

**Оригинал:** [Trail: The Extension Mechanism](https://docs.oracle.com/javase/tutorial/ext/index.html)

Как сделать пользовательские API доступными всем приложениям на платформе Java.

> **Примечание.** Механизм расширений (`lib/ext`, `java.ext.dirs`) объявлен устаревшим в JDK 8
> (JEP 220) и удалён в JDK 9+; его заменила система модулей (JPMS). Механизм `ServiceLoader`/SPI
> при этом сохранился. Материал приведён как перевод оригинала.
