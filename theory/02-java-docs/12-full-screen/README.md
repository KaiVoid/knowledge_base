# Трейл 12. Full-Screen Exclusive Mode API (Полноэкранный эксклюзивный режим)

**Оригинал:** [Trail: Full-Screen Exclusive Mode API](https://docs.oracle.com/javase/tutorial/extra/fullscreen/index.html)

API полноэкранного эксклюзивного режима (появился в JDK 1.4) для высокопроизводительной графики
и игр: управление режимом отображения, активный рендеринг, двойная буферизация и переключение
страниц.
