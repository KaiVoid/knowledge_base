# Трейл 14. Internationalization (Интернационализация)

**Оригинал:** [Trail: Internationalization](https://docs.oracle.com/javase/tutorial/i18n/index.html)

Как проектировать программу так, чтобы её можно было легко адаптировать (локализовать) к разным
языкам и регионам: локали, ресурсные пакеты, форматирование чисел/дат/сообщений, работа с
текстом и поставщики служб.
