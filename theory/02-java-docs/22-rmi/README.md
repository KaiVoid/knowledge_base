# Трейл 22. RMI (Remote Method Invocation)

**Оригинал:** [Trail: RMI](https://docs.oracle.com/javase/tutorial/rmi/index.html)

Удалённый вызов методов (RMI) позволяет объекту вызывать методы объекта, работающего на другой
виртуальной машине Java. Обзор RMI-приложений и полный пример «вычислительный движок».
