# Правила оформления раздела «Java-документация»

Раздел — это перевод официального руководства **The Java Tutorials (Oracle)** на русский.

## Структура

- Один трейл = один каталог `NN-<trail-slug>/`. Один урок = один файл `NN-<lesson-slug>.md`.
- В каталоге трейла обязателен `README.md`: обзор трейла, ссылка на оригинал, список уроков
  со статусом (✅ переведён / 🚧 в работе / ⬜ запланирован).
- Сквозная нумерация уроков внутри трейла. Подтемы оформляются как `##`/`###` внутри файла.

## Контент

1. **Язык — русский.** Переводится весь текст оригинала. Ничего не выдумывается:
   материал соответствует исходной странице Oracle.
2. **Достоверность.** В конце каждого файла секция `## Источник` с точной ссылкой на
   страницу Oracle (рабочий URL). Если урок собран из нескольких страниц — перечислить все.
3. **Код примеров** сохраняется как в оригинале (Java — язык-независим). **Комментарии
   внутри кода переводятся на русский.** Имена классов/методов/переменных не меняются.
4. **Термины.** Общепринятые англоязычные термины (garbage collector, stream, thread)
   даются по-русски с пояснением, при первом упоминании — с оригиналом в скобках.

## Графика и диаграммы

- **Векторные диаграммы** (иерархии, потоки, схемы) пересоздаются в **Mermaid** с русскими
  подписями — блок ` ```mermaid `. Это предпочтительный способ.
- **Растр**, который Mermaid не выражает, перерисовывается с русским текстом и кладётся в
  `../assets/<trail>/`, подключается через `![подпись](../assets/<trail>/file.png)`.
- У каждого изображения — русская подпись и при необходимости текстовое описание содержимого.
- Исходные англоязычные картинки можно сохранить рядом с пометкой «оригинал», если перевод
  ещё не готов (вариант для скоупа «исходник + переведённая подпись»).

## Перекрёстные ссылки

- На смежные области базы знаний — через `[[NN-slug]]` (файлы `theory/01-knowledge-areas/`).
- В шапке урока — ссылка на соответствующий раздел вопросов `interview-questions/`.

```markdown

**Трейл:** <название трейла> · **Оригинал:** [<страница>](<URL Oracle>)
**Связанные области:** [[NN-slug]] · **Вопросы:** <раздел>

> Перевод официального руководства Oracle.

## <Подтема>

<перевод; диаграммы Mermaid; примеры кода с переведёнными комментариями>

## Источник

- [<точное название страницы>](<URL>) — официальное руководство Oracle.
```
