# Веб-просмотрщик базы знаний

Удобный просмотр базы вопросов и теории в браузере. Один файл, **без сторонних
зависимостей** — нужен только Python 3.

## Запуск

```bash
cd webapp
python3 app.py          # порт 8000 по умолчанию
python3 app.py 9000     # или указать свой порт
```

После старта приложение само откроет браузер на `http://127.0.0.1:8000/`
(если не открылось — перейдите по адресу вручную). Остановка — `Ctrl+C`.

## Возможности

- **Две вкладки:** «Вопросы» и «Теория».
- **Поиск** по тексту вопросов и ответов (по всей базе или внутри раздела).
- **Фильтры:** по разделу (боковая панель) и по уровню (Junior / Middle / Senior).
- **Карточки вопросов:** клик раскрывает обе секции — «Оригинальный ответ из интернета»
  (со ссылкой на источник) и «Ответ от Claude».
- **Режим самопроверки:** галочка «Скрыть ответы» размывает ответы; по кнопке «Показать ответ»
  открывается конкретный.
- **Подсветка кода** Java и SQL, рендеринг markdown-таблиц.

## Как устроено

`app.py` при старте читает `../theory/**/*.md` и `../interview-questions/**/*.md`,
парсит их по правилам проекта (заголовок `### Вопрос N.`, секции ответов, источник, уровень),
рендерит markdown в HTML собственным компактным конвертером и отдаёт через локальный HTTP-сервер.
Данные только для чтения; при добавлении новых вопросов достаточно перезапустить сервер.

Эндпоинты: `/` (страница), `/api/index`, `/api/q/<id>`, `/api/search`, `/api/theory`, `/api/theorydoc`.
